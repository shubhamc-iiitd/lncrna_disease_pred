Easiest — one command (download, ingest, train if needed)

cd {{ root }}
pip install -r requirements.txt
python run_flask.py --auto-setup

If data/associations.csv is missing, this fetches LncRNADisease v3.0 and builds the CSVs. If checkpoints/lightgcn_full.pt is missing and PyTorch is installed, it trains LightGCN (can take several minutes). Same behavior: export LNC_AUTO_SETUP=1 then python run_flask.py.

Data-only (no training): python run_flask.py --auto-setup --skip-train

Manual — full LncRNADisease v3.0 graph under data/

cd {{ root }}
python scripts/fetch_lncrnadisease_v30.py
python scripts/ingest_lncrnadisease_v30.py
python scripts/train_lightgcn_full.py --data-dir data
python run_flask.py

The app uses data/ automatically when those CSVs exist (no env var needed).

Fallback — bundled demo

If data/associations.csv is missing, the app falls back to examples/minimal_data/. Or set LNC_DATA_DIR to any folder with the three CSVs.
